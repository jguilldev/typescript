CURSO DE TYPESCRIPT

Modo observador

El modo observador lo que hace es estar pendiente de los cambios para actualizar de forma automatica

para ello desde la terminal se usa el comando de siempre "tsc script.ts" pero se le agrega -w al final

Asi cada vez que guardemos el cambio se refleja en el archivo.js

Inicializar Typescript

Para inicializar Typescript y que nos de caracteristicas avanzadas

se debe iniciar como se hace con git o node se usa "tsc - init"
